# Release Notes - v1.2.0

## Added
- 타일/목록 보기 이름 바꾸기를 팝업 입력 방식에서 인라인 편집 방식으로 통일했습니다.
- 즐겨찾기 파일/폴더 오버레이의 기본 트리 확장 표시를 개선했습니다.
- 파일 목록 대량 표시 시 성능 개선을 위한 배치 처리 컬렉션(`RangeObservableCollection`)을 추가했습니다.
- 앱 아이콘을 `pro_icon.ico`로 교체해 실행 파일/창 아이콘을 통일했습니다.

## Changed
- 즐겨찾기 오버레이를 별도 플로팅 창 중심 방식에서 메인 창 내 오버레이 중심으로 조정해 끊김을 줄였습니다.
- 트리 노드 확장 가능 여부 계산을 실 디렉터리 기준으로 정밀화했습니다.
- 메모 툴팁 위치/패딩을 조정해 가독성을 개선했습니다.
- 폴더 이동 후 기본 선택 항목을 상위 폴더(`[..]`)로 맞추는 동작을 강화했습니다.

## Fixed
- 파일 이름 바꾸기 후 Enter 입력 시 앱이 종료되던 문제를 수정했습니다.
- 일부 하위 폴더가 없어도 트리에서 확장 아이콘이 보이던 문제를 수정했습니다.
- 외부 파일 변경(예: 다운로드 완료) 후 목록 갱신이 늦게 반영되던 케이스를 보완했습니다.
- 이미지 뷰어 Fit/확대 축소 동작에서 스크롤/표시 비율이 어색하던 부분을 개선했습니다.

## Build
- Target: `net8.0-windows`
- Runtime: `win-x64`
- Version: `1.2.0`

# Release Notes - v1.1.0

## Added
- 즐겨찾기 파일/폴더 UI를 고정 오버레이 트리 방식으로 개선해, 팝업 전환 시 체감 지연을 최소화했습니다.
- 즐겨찾기 폴더 트리를 기본 확장 상태로 열어 첫 탐색 진입성을 높였습니다.

## Changed
- 즐겨찾기 관련 인터랙션을 클릭 중심으로 단순화해 마우스 이동 시 불필요한 이벤트 처리 비용을 줄였습니다.
- 탭 전환/즐겨찾기 표시 경로의 계측 로그를 정리하고, 기본 실행 성능을 우선하도록 설정했습니다.

## Fixed
- 즐겨찾기 파일 창을 한 번 연 뒤 탭/폴더 이동이 느려지던 문제를 완화했습니다.
- 플로팅 표시가 환경에 따라 비정상 위치에 표시되거나 보이지 않던 케이스를 수정했습니다.
- 삭제/탭 전환 이후 발생하던 불필요한 정리 루틴 호출을 줄여 UI 끊김을 개선했습니다.

## Build
- Target: `net8.0-windows`
- Runtime: `win-x64`
- Publish: framework-dependent (`--self-contained false`)

# Release Notes - v1.3.0

## Added
- 패널 하단 임베디드 터미널 연동 기능을 추가했습니다.
- `Ctrl+R`로 파일리스트 내 검색 입력 박스에 즉시 포커스 이동하는 단축키를 추가했습니다.
- 상단 즐겨찾기 버튼에 폴더 아이콘을 표시해 가독성을 개선했습니다.

## Changed
- 목록/간단목록/상세보기 삭제 동작을 정비해 보기 모드별 동작 차이를 줄였습니다.
- 상세보기에서만 `좌/우` 화살표로 탭 이동이 동작하도록 키보드 동작을 조정했습니다.
- 파일 삭제 후 선택 위치를 맨 위가 아니라 삭제한 항목 주변 위치로 유지하도록 개선했습니다.
- 이미지 뷰어 기본 표시를 Fit 활성화 상태로 변경했습니다.

## Fixed
- 터미널 입력에서 `Space`/`Backspace` 키 처리 및 커서 표시 문제를 보완했습니다.
- 목록보기/간단목록보기에서 삭제 시 동작 실패 및 확인창 중복 노출 문제를 수정했습니다.
- 이미지 뷰어에서 `Del` 키 삭제가 동작하지 않던 문제를 수정했습니다.
- 검색(파일 찾기) 창에서 요청된 UI 항목(플러그인 불러오기/저장, 동작 보기 결과 옵션) 노출을 정리했습니다.
- 인라인 이름변경 처리의 불필요한 비동기 선언을 정리해 `CS1998` 빌드 경고를 제거했습니다.

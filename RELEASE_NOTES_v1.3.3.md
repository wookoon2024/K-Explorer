# Release Notes - v1.3.3

## Fixed
- 인라인 이름 변경(F2) 후 선택/포커스가 `[..]` 또는 다른 항목으로 튀는 문제를 수정했습니다.
- 이름 변경 후 목록 스크롤이 맨 위로 이동하거나, 다음 행으로 내려갔다가 복귀하는 문제를 수정했습니다.
- 정렬(예: 날짜 정렬) 상태에서 키보드 `위/아래` 이동이 화면 순서와 다르게 동작하던 문제를 수정했습니다.
- 패널 파일 목록에서 `Backspace` 입력 시 상위 폴더로 이동하던 동작을 제거했습니다.
- 좌/우 화살표로 탭 전환 또는 반대 패널로 포커스가 이동하던 동작을 제거했습니다.
- 스크롤바 조작 중 드래그 오인으로 금지 커서(`⊘`)가 뜨며 스크롤이 방해되던 문제를 수정했습니다.

## Changed
- 폴더 진입/상위 이동 시 선택 기준을 일관화하여 `[..]` 선택 동작을 개선했습니다.
- 파일 충돌 처리 UX를 개선했습니다.
  - 드래그 복사/이동 시 자동 이름 변경 대신 충돌 확인 팝업을 표시합니다.
  - `Ctrl+C` / `Ctrl+V` 붙여넣기에도 동일한 충돌 확인 팝업을 적용했습니다.
  - 충돌 팝업 한글 문구 표시를 안정화했습니다.

# Release Notes - v1.3.2

## Added
- 도구모음 아이콘을 SVG/벡터 리소스로 교체해 해상도 독립 렌더링 품질을 개선했습니다.
- 도구모음 아이콘 호버/선택 상태(하늘색 계열) 시각 피드백을 추가했습니다.

## Changed
- 도구모음 아이콘 간 간격과 배경 처리를 조정해 가독성과 일관성을 높였습니다.
- 목록/타일 전환 시 패널 상단(드라이브/탭)과 본문 폭 동기화 기준을 정리했습니다.
- 터미널 커서를 깜빡임 타이머 기반에서 고정 블록 커서(`█`) 기반으로 변경했습니다.

## Fixed
- 즐겨찾기 파일/폴더 플로팅 패널이 포커스 이동 시 닫히지 않던 문제를 수정했습니다.
- 일부 이미지 파일 미리보기가 표시되지 않던 케이스에 대해 디코딩 폴백 경로를 보완했습니다.
- 대용량 파일 리스트에서 체감 지연이 커지던 경로를 완화하도록 열거/정렬 처리 흐름을 개선했습니다.

## Build
- Target: `net8.0-windows`
- Runtime: `win-x64`
- Version: `1.3.2`
- Release date: `2026-04-27`

# Release Notes - v1.3.1

## Added
- LoadPanel 상세 성능 로그를 `fetch/prioritize/bind` 단계로 분리하여 병목 파악을 쉽게 했습니다.
- 최신 버전/최신 다운로드 링크를 README에 추가했습니다.

## Changed
- 폴더 로딩 타임아웃 처리를 `Task.Delay` 대기 방식에서 취소 토큰 기반으로 변경해 불필요한 백그라운드 작업을 줄였습니다.
- 패널 아이템 우선순위/필터/요약 계산 경로를 단일 패스 중심으로 정리하여 UI 스레드 부담을 낮췄습니다.
- 탭 전환/선택 변경 시 중복 이벤트 처리와 과도한 트레이스를 줄여 반응성을 개선했습니다.
- 즐겨찾기/핀 토글 이후 전체 패널 재로딩 대신 영향 경로 위주 재로딩으로 범위를 축소했습니다.
- 빠른 접근/대시보드 갱신 시 파일 존재/수정시각 조회 캐시를 적용했습니다.

## Fixed
- 사용 이력 저장 시 전체 삭제 후 재기록하던 경로를 dirty upsert 방식으로 전환해 저장 지연을 줄였습니다.
- 설정 저장 시 전체 재기록 대신 변경분 중심 저장으로 전환해 불필요한 DB 작업을 줄였습니다.

- Runtime: `win-x64`
